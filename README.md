# 🔧 АвтоПромпт-Мастер (AutoPrompt Master)

**Автономный HTML-инструмент для генерации мощных диагностических промптов.  
10 шаблонов, встроенные подсказки, чипсы симптомов — скопируй, вставь в ChatGPT/Claude/Ollama и получи точный план ремонта.**

![AutoPrompt Master](https://img.shields.io/badge/version-1.0-yellow) ![License](https://img.shields.io/badge/license-MIT-blue)

## 🚀 Возможности

- **10 предустановленных шаблонов** – двигатель, трансмиссия, ходовая, тормоза, электрика, кузов, климат, топливная, выхлоп, общая диагностика.
- **Умная сборка промпта** – поля «Роль», «Задача», «Контекст», «Правила» заполняются автоматически при выборе шаблона.
- **Контекстные подсказки** – кнопки `💡` раскрывают примеры марок, моделей, список симптомов.
- **Чипсы симптомов** – кликни по симптому под полем ввода, и он добавится в текст. Не нужно печатать «Check Engine, троит, стук…» — собирай кликами.
- **Цепочка рассуждений и безопасность** – две опциональные галки добавляют в промпт инструкции «Думай шаг за шагом» и «Укажи критические риски».
- **Копирование в один клик** – готовый промпт сразу в буфер обмена.
- **Полностью локальный** – никаких серверов, ничего не отправляется в сеть. Открыл `auto-prompt.html` в браузере и работай.
- **Адаптивный интерфейс** – тёмная тема, удобно на ПК и планшете.

## 🛠️ Как использовать

1. **Скачай** файл `auto-prompt.html` из этого репозитория.
2. **Открой** его в любом современном браузере (Chrome, Firefox, Edge).
3. **Выбери шаблон** диагностики (Двигатель, Трансмиссия и т.д.).
4. **Заполни марку, модель и симптомы** – используй подсказки `💡` и кликабельные чипсы для быстрого ввода.
5. **Нажми «⚡ Собрать промпт»** – в нижнем блоке появится готовый текст.
6. **Скопируй** его кнопкой «📋 Копировать» и вставь в ChatGPT, Claude или любую другую LLM.
7. **Получи структурированный план диагностики** с вероятными причинами, способами проверки и мерами безопасности.

## ✨ Пример сгенерированного промпта

Вот как выглядит результат для шаблона **«Двигатель»** с симптомами «Check Engine, троит на холодную, потеря мощности» для автомобиля Toyota Camry 2019 2.5 л:

```
РОЛЬ: Ты — диагност по двигателям внутреннего сгорания с 20-летним стажем. Специализируешься на механических неисправностях, системе зажигания, ГРМ, датчиках.

ЗАДАЧА: Проведи пошаговую диагностику неисправности двигателя. Назови 3-5 вероятных причин, для каждой — способ проверки (визуальный, мультиметр, сканер OBD2) и ожидаемые параметры.

КОНТЕКСТ: Автомобиль: Toyota Camry 2019, 2.5 л. Симптомы: Check Engine, троит на холодную, потеря мощности.

ПРАВИЛА: 1. Исключи критические риски (перегрев, падение давления масла).
2. Проверь коды ошибок OBD2.
3. Для каждой гипотезы укажи конкретные цифры (компрессия, напряжение датчиков).
4. Если нужна разборка узла — предупреди.

ЦЕПОЧКА РАССУЖДЕНИЙ: Думай шаг за шагом, объясняй каждый этап.

БЕЗОПАСНОСТЬ: Отметь все шаги, связанные с риском для жизни (высокое напряжение, топливная система, подушки безопасности). Если симптом критический — порекомендуй немедленно прекратить эксплуатацию.
```

Отправьте этот текст в любую нейросеть — и вы получите подробный план проверок, а не общий совет «поменяйте свечи».

## ❓ Часто задаваемые вопросы

**1. Нужен ли интернет для работы инструмента?**  
Нет. Всё работает локально в вашем браузере. Интернет понадобится только если вы решите вставить готовый промпт в онлайн-сервис (ChatGPT, Claude).

**2. Могу ли я добавить свои шаблоны?**  
Да. Отредактируйте объект `templates` в JavaScript-коде файла, как описано в разделе «Добавление своих шаблонов».

**3. Можно ли использовать инструмент на мобильном телефоне?**  
Да. Интерфейс адаптивный. Откройте `auto-prompt.html` в мобильном браузере.

**4. Какие данные я должен ввести в поле «Симптомы»?**  
Максимально подробные: когда появляется симптом, на каком пробеге, что предшествовало. Например, не просто «стук», а «стук в передней подвеске при проезде лежачих полицейских на скорости 20 км/ч». Используйте кликабельные чипсы для базовых симптомов.

**5. Что дают галочки «Цепочка рассуждений» и «Безопасность»?**  
Они добавляют в промпт специальные инструкции. «Цепочка рассуждений» заставляет нейросеть объяснять ход мысли, а «Безопасность» — предупреждать об опасных операциях.

**6. Могу ли я использовать этот промпт для диагностики электромобиля?**  
Частично. Текущие шаблоны ориентированы на ДВС, но вы можете создать свой шаблон для электрокара, добавив высоковольтную безопасность и диагностику батареи.

**7. Куда вставлять сгенерированный промпт?**  
В ChatGPT, Claude, Gemini, локальные модели через Ollama, или в любой другой интерфейс LLM.

**8. Чем ваш инструмент лучше простого вопроса «Что с моей машиной?»**  
Стандартный запрос даст общий ответ. Наш промпт превращает вас в эксперта, задающего правильные рамки: роль специалиста, контекст, правила, обязательную проверку безопасности. Результат — точный план, а не догадки.

**9. Могу ли я использовать инструмент в коммерческих целях?**  
Да, лицензия MIT разрешает любое использование, включая коммерческое.

**10. Как часто обновляются шаблоны и подсказки?**  
Вы можете сами их актуализировать, редактируя объект `templates` в исходном коде. Если появятся новые версии от автора, они будут опубликованы здесь же на GitHub.

## 🧩 Структура файла

Весь код в одном HTML-файле:

- **HTML** – интерфейс с выпадающим списком шаблонов, полями ввода, кнопками.
- **CSS** – тёмная тема, адаптивная вёрстка.
- **JavaScript** – логика сборки промпта, база знаний шаблонов, управление подсказками и чипсами.

Никаких зависимостей, фреймворков или сборщиков. Один файл — и всё работает.

## ✏️ Добавление своих шаблонов

Шаблоны хранятся в объекте `templates` внутри `<script>`. Чтобы добавить новый (например, «Рулевое управление»):

1. Открой `auto-prompt.html` в любом редакторе.
2. Найди объект `templates` и добавь новое свойство:

```javascript
steering: {
  role: "Ты — эксперт по рулевому управлению...",
  task: "Диагностируй проблему...",
  context: "Автомобиль: [Марка] [Модель]...",
  rules: "1. Проверь...",
  hints: {
    template: "🔧 Рулевое: стуки, люфт, тяжёлый руль...",
    brand: "Примеры: BMW, LADA...",
    model: "Например: «Vesta», «3 series E90»...",
    symptoms: ["люфт руля", "стук в рулевой рейке", "тяжело крутить", "гул насоса ГУР"]
  }
}
```

3. Добавь `<option value="steering">Рулевое управление</option>` в `<select id="templateSelect">`.
4. Сохрани файл — новый шаблон готов.
